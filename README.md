# Telegram SMM Bot

## Features
- Admin Password Login
- Facebook Link Verification
- Service Selection
- Quantity Input
- Order Submit
- Balance Check
- Vercel Ready

## Deploy
1. Upload to GitHub
2. Import to Vercel
3. Add Environment Variables
4. Set Telegram Webhook

## Webhook URL
https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=https://YOUR_DOMAIN.vercel.app/api/webhook
